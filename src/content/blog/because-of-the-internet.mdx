This post has a `post_hero.date` set in the future. Until a build runs on or after that date, it is excluded from the build entirely — it has no page, and it does not appear in the blog index, tag pages, the RSS feed, or any "Recent Posts" list.

When the date is in the future, `scripts/generate-schedule.mjs` adds a line for this post to `_schedule.txt` in the build output. CloudCannon reads that file and schedules a build at the timestamp. Once that build runs, the date has passed, the publish gate in `src/utils/posts.ts` lets the post through, and it goes live — no manual step required.

Change the date to the past and rebuild to see it appear immediately.
